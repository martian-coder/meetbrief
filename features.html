MeetBrief / Features

Every feature you need to
never be caught off guard

MeetBrief combines local speech recognition, streaming AI responses, and an unobtrusive overlay into a single tool that works with any meeting platform.

Start your free trial

Real-time transcription — hands free

Whisper runs locally on your machine, capturing the other person's audio the moment they speak. A rolling 30-second transcript window is built and maintained automatically — no cloud API required, no latency waiting on a server round-trip. The model detects end-of-turn boundaries using Silero VAD, so it knows exactly when to generate a reply.

AI-suggested replies, streamed instantly

The moment the speaker pauses (~1 second of silence), the LLM is fired. Responses stream token-by-token to the overlay so the first words appear in under a second. Context-aware and grounded in your pre-loaded meeting notes.

Transparent always-on-top overlay

A frameless, click-through window floats above every app. Glance at it while you speak. Toggle visibility, click-through mode, or force-generate with keyboard shortcuts at any time.

Pluggable LLM & STT providers

Switch between Claude API, Ollama, or other providers with a single config change. Prompt caching keeps costs low on Anthropic's API by caching your static context. Use Ollama for fully offline operation.

100% local. Zero cloud dependency

Audio never leaves your machine. Transcription runs on-device via Whisper. AI calls go directly from your machine to the API provider you choose — MeetBrief has no relay servers, no analytics, no telemetry.

Multilingual — any language, English output

Whisper's translate task converts speech in any of 99 supported languages to English text automatically. Works for multilingual meetings with Mandarin, French, Spanish, Arabic, and more speakers at once — the AI always works in English regardless of what's spoken.

Integrations

Your providers, your choice

MeetBrief is provider-agnostic. Switch between cloud and local with a single config value.

C

Anthropic Claude

claude-opus-4-8 via the Messages API with prompt caching on your static context. Fastest time-to-first-token, highest quality responses for nuanced conversations.

🦙

Ollama (local)

Any model running on localhost:11434 via the OpenAI-compatible endpoint. Works completely offline — no API key, no cost per call. Good for air-gapped or privacy-sensitive environments.

W

Whisper (local STT)

Runs entirely on-device via @huggingface/transformers. Silero VAD handles turn detection. Translate mode converts any of 99 languages to English automatically.

Keyboard shortcuts

Stay in flow with hotkeys

Every action has a keybind. Glance, trigger, dismiss — all without touching your mouse.

⌨️

Toggle overlay visibility

Show or hide the floating suggestion panel instantly. The overlay is always rendered but can be toggled invisible so it never distracts when you don't need it.

⚡

Force-generate on demand

Don't wait for a silence trigger. Press the hotkey at any moment to send the current transcript window to the LLM and get a fresh suggestion immediately.

🖱️

Click-through mode

Toggle mouse events on the overlay so clicks pass through to the app behind it. When you want to interact with the overlay (scroll, select text), toggle it back.

🔧

Fully customizable

All keybinds are configurable in the Settings panel. Change any shortcut to avoid conflicts with your meeting app or OS-level shortcuts.
